# Beginners Gulp Boilerplate

Simple gulp boilerplate with gulp tasks as:

  - browserify
  - eslint
  - sass
  
### Installation


```sh
$ npm install
```

##### Production build
For the minified versions of all files, run
```sh
$ gulp
```
